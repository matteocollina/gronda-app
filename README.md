# Example Gronda App

## 1 - How to install dependencies
`
$ npm install
`\
`
$ cd ios && pod install
`

### 1a - How to run iOS app
`
$ npm start
`\
`
$ npx react-native run-ios
`

### 1b - How to run Android app
`
$ npm start
`\
`
$ npx react-native run-android
`

## 2 - How to run tests
`
$ npm run test
`
